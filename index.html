Сделать скрипт-планировщик публикаций в ТГ-группу

Цена: 3 000 ₽

Добрый день. Есть несколько своих Телеграм-групп. Различной тематики. Живой чат с подписчиками. Есть подготовленный контент (картинки, музыка, клипы, ролики, опросы), загруженный в ТГ-каналы по тематикам. Необходимо: 1. Вывести управление группами и каналами-библиотеками в вэб-админку. БД и админку (на Бутстрап) делаем с нуля. Бота с правами админа - добавлю в нужные чаты. 2. Настроить планировщик задач (например, через Cron и т.п.) - чтобы нужные публикации из нужного канала в нужное время и день публиковались в нужной группе. Добавление новой задачи. Синхронизировать время публикаций. 3. Оптимизировать скрипт. Доработать удобство и оформление страниц админки. Опубликовать проект на VDS. Прошу указывать Ваш опыт работы с ТГ-ботами. Стэк - предпочтительнее C#. Фреймворк для бэкенда - ASP.NET Core Web Api, в качестве ORM - Entity Framework Core, для тг ботов - TelegramBot, UserBot

Голосование под видео в тг

Цена: 10 000 ₽

Коллеги доброго дня! Требуется организовать голосование по мере просмотра видео/трансляции в телеграме - чтобы выступающие в дискуссии могли обращаться к зрителям с вопросами - а зрители - голосовать по мере просмотра - и смотреть общий график голосов! Напишите пожалуйста ваше видение реализации и примерную смету в часах и свой тариф! Наше видение - что это может быть телеграм-бот который по команде запрашивает от пользователя видео - пользователь аплоадит или пересылает боту видео - бот проигрывает видео и под ним показывает кнопки согласен/непонятно/несогласен - при нажатии на которые - на внешний апи отправляется событие - юзерид|видеоид*|время. Результаты показывать в обновляемом сообщении - в таблице - получаемой из внешнего апи - время|суммасогл|сумманепон|сумманесогл. *Видеоид у пересылаемого видео должно совпадать - чтобы суммировать по нему голоса! Возможно это удобнее сделать как web app или как game - в html5 - тогда результат можно показать в виде графика? Напишите пожалуйста своё видение решения этой задачи - задача не последняя - мы нацелены на долгосрочное сотрудничество!
